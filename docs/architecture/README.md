### Mô tả
- Hệ thống được thiết kế theo hướng API-first, dễ dàng mở rộng cho web/mobile.
- Sử dụng các service tách biệt: Backend, Frontend, Payment, Message Queue, Worker.
- Đảm bảo khả năng mở rộng, bảo mật, và hiệu năng cao.

Bạn có thể cập nhật thêm các diagram hoặc mô tả chi tiết vào thư mục này bất cứ lúc nào.
