# API Documentation

## Tổng quan
Tài liệu này sẽ mô tả các endpoint API chính của hệ thống Nike Clothing Store E-Commerce Platform.

---

## Danh sách các nhóm API
- User Authentication & Authorization
- Product Catalog
- Shopping Cart
- Checkout & Payment
- Order Management
- Shipping Tracking
- Admin Dashboard

---

Bạn có thể bổ sung chi tiết từng endpoint, ví dụ:

### [GET] /api/products
- Trả về danh sách sản phẩm

### [POST] /api/orders
- Tạo đơn hàng mới

...v.v.

Bạn có thể cập nhật thêm các file hoặc chi tiết endpoint vào thư mục này bất cứ lúc nào.
